# 用神体系

用神是八字中最关键的概念——对命局最有利的那个五行/十神。找准用神，方能准确判断吉凶祸福。

## 一、用神取法四大体系

### 1. 扶抑法（平衡法）— 最常用

日主强旺 → 用神取克泄耗（官杀、食伤、财）
日主衰弱 → 用神取生扶（印、比劫）

**判断日主强弱**:
- 得令（生旺月）：+40%
- 得地（地支通根）：+30%
- 得生（印星贴身）：+20%
- 得助（比劫帮扶）：+10%

**日主强弱等级**:
- **太强**: 用官杀克，或食伤泄，不宜再用印比
- **偏强**: 用财官食伤
- **中和**: 随运变化
- **偏弱**: 用印比
- **太弱（从格）**: 不能扶，顺其势

### 2. 调候法 — 穷通宝鉴体系

寒暖燥湿的平衡。日主生于不同月份，因气候不同而有不同需求。

**调候原则**:
- **冬生（亥子丑月）**: 天寒地冻，急须火调候，其次木生火。水冷木不发，无火则万物不生
- **夏生（巳午未月）**: 火炎土燥，急须水调候，其次金生水
- **辰戌丑未月**: 土当令而燥（未戌）或湿（辰丑），亦须调候
- **春生**: 寒暖交接，视具体命局。春初（寅月）仍须火暖
- **秋生**: 金当令而寒，须火暖金方能成器

**穷通宝鉴逐月调候精要**（详见 classical-texts/qiongtongbaojian.md）:

以日主为甲木为例：
- 寅月甲木：寒木向阳，须火暖之
- 卯月甲木：春木当令，有水水火既济最佳
- 辰月甲木：木有余气，火土并用
- 巳月甲木：夏木枯燥，急须水润
- 午月甲木：炎炎夏日，非水不可
- 未月甲木：土燥木枯，水为先
- 申月甲木：秋金克木，先丁火制金，次水润木
- 酉月甲木：同上，更须火制金
- 戌月甲木：燥土，先水后火
- 亥月甲木：寒木，须火暖
- 子月甲木：极寒，火土并用
- 丑月甲木：寒土，火为先

**各日主通用调候**:
- **甲乙木**: 夏须水，冬须火
- **丙丁火**: 夏须水制其炎，冬须木生其身
- **戊己土**: 夏须水润，冬须火暖
- **庚辛金**: 夏须水淘，冬须火煅
- **壬癸水**: 冬须火防冻，夏须金生水

### 3. 通关法

两种五行相战，取通关之五行为用。

| 相战 | 通关五行 |
|------|----------|
| 木土交战 | 火 |
| 土水交战 | 金 |
| 水火交战 | 木 |
| 火金交战 | 土 |
| 金木交战 | 水 |

> 例：食伤（木）制杀（土）太过 → 用火通关，木生火，火生土，化敌为友

### 4. 病药法 — 神峰通考体系

找出命局中最突出的"病"，用神即为"药"。

- 太旺 → 泄耗为药
- 太弱 → 生扶为药
- 官杀混 → 去其一为药
- 寒 → 火为药
- 燥 → 水为药

## 二、喜神、忌神、仇神、闲神

| 类型 | 定义 | 作用 |
|------|------|------|
| **用神** | 对命局最利的五行/十神 | 扶助用神则吉 |
| **喜神** | 生助用神者 | 喜神出现，用神得力 |
| **忌神** | 克制用神者 | 忌神当运，灾祸立至 |
| **仇神** | 用神克之而反受伤，或生忌神克用神者 | 仇神助纣为虐 |
| **闲神** | 对用神无显著影响者 | 中性的存在 |

**用神与忌神的辩证关系**:

1. **用神不独一**: 有时用神有两个甚至三个（如既需火调候又须水润局，水火并用）
2. **用神会随大运变化**: 某运中用神得力，另一运中用神失势 → 用神不是永恒的
3. **忌神有用**: 忌神过旺受制反为权（如七杀忌神被食神克制 → 武贵）；忌神制忌神也是救应
4. **闲神不闲**: 闲神在大运中可能转化为喜神或忌神
5. **过犹不及**: 用神过旺也会致病（如用印太过则依赖成性）；用神需要"恰到好处"

### 用神的层次与真假

**用神五个层次**:

| 层次 | 条件 | 效果 |
|------|------|------|
| **真用神** | 得令 + 透干 + 通根 + 无克破 | 大富贵，功成名就 |
| **有用之神** | 透干或通根 + 无克破 | 小康以上，安稳顺遂 |
| **虚用神** | 透干但无根（虚浮），不被克 | 虽有如无，心高命薄 |
| **受损用神** | 用神被克、被合、被泄 | 怀才不遇，处处受阻 |
| **假用神** | 看似为用，实则被暗伤 | 先成后败，虚花一场 |

**如何辨别用神真假**:

1. **得令**: 用神五行在月令当旺 → 力量最大
2. **得地**: 用神在地支有根（长生/禄/旺/库） → 根基扎实
3. **透干**: 用神在天干显豁 → 力量能发挥
4. **无伤**: 用神不被克、合、刑、冲 → 不受干扰
5. **有源**: 有喜神生助用神 → 源远流长

> **最怕**: 用神被忌神合去（用神失其用）、用神入墓（有力无处使）、用神逢冲（根基动摇）

### 扶抑法深度应用

**日主太强**（月令旺 + 地支3个以上根 + 天干印比多）:
- 首选 **官杀** 克之 → 但须官杀有根，否则如以卵击石
- 次选 **食伤** 泄之 → 泄秀生财，才华致富
- 再用 **财星** 耗之 → 但身太强不忌财，反能发财
- **忌**: 再用印比（火上浇油）

**日主偏强**（月令旺 + 1-2个根):
- 首选 **财官** 并用 → 财生官，官护财
- 次选 **食伤生财** → 技艺致富
- **忌**: 再遇比劫运（夺财）

**日主偏弱**（月令休囚 + 只有1个弱根):
- 首选 **印星** 生扶 → 以印化杀生身
- 次选 **比劫** 帮身 → 但比劫帮身也夺财
- **忌**: 再遇财官运（克泄交加）

**日主太弱**（月令死绝 + 无根 + 无生扶）→ **从格**:
- 不能生扶！只能顺着旺势走
- 从官杀 → 喜财生杀
- 从财 → 喜食伤生财
- 从儿 → 喜财泄食伤之气

### 调候法深入应用

调候之所以重要，是因为寒暖燥湿是命局的根本气质。格局再好，若寒无火暖，万物不生；格局再差，有火调候，生机盎然。

**十干最根本的调候需求**:

| 天干 | 最需要的调候 | 原因 |
|------|-------------|------|
| 甲 | 冬须火，夏须水 | 参天大树，寒则不发，燥则枯槁 |
| 乙 | 冬须火，夏须水 | 花草更娇，寒暖对其影响最大 |
| 丙 | 冬须木助火势，夏须水济火炎 | 太阳之火，但夏火过烈须水 |
| 丁 | 冬须甲木为源，夏须壬水调和 | 灯烛之火，冬无木则熄，夏无水则燃 |
| 戊 | 冬须火暖，夏须水润 | 厚重之土，冻则无用，焦则不生 |
| 己 | 冬须火，夏须水 | 田园之土，同样需要温润 |
| 庚 | 任何时节都需火炼 | 顽金无火不成器，四季皆宜火炼 |
| 辛 | 冬须火暖，热则喜壬 | 珠宝之金，寒则无光泽 |
| 壬 | 冬须火防冻，夏须金生源 | 江河之水，冻则止流 |
| 癸 | 冬须火暖，夏须金生 | 雨露之水，冬凝夏涸 |

**调候的三层判断**:
1. **季节**: 冬→火, 夏→水, 辰戌丑未→看燥湿
2. **日主五行**: 木冬喜火夏喜水, 金四季喜火
3. **具体八字**: 八字已有调候五行则无须外求；虽寒但水多则不宜再用火

## 三、用神有力判断

用神须满足以下条件方为有力：

1. **得令**: 用神五行在月令当旺 — 力量40%
2. **得地**: 用神在地支有根（长生/禄/旺/库） — 力量30%
3. **得生**: 有喜神生助用神 — 力量20%
4. **透干**: 用神在天干透出，力量能发挥 — 力量10%
5. **不受克**: 用神不被忌神克制、合住、冲散

> 用神虚浮无根（仅在天干而无地支支撑），如纸老虎，虽有如无。
> 用神藏而不透（仅在地支而无天干引出），待大运引出方显其用。
> 用神透干通根且得令 → **真用神**，大富贵之基。
> 用神被合（如丙用神被辛合）→ 用神失其用，先吉后凶或有力无处使。

### 用神被制的常见情形

| 情形 | 影响 | 举例 |
|------|------|------|
| 被合 | 用神失其用 | 辛为用神，被丙合（丙辛合水） |
| 被克 | 用神受伤 | 木为用神，被申酉金直克 |
| 被冲 | 根基动摇 | 用神地支被冲（寅申冲、卯酉冲） |
| 入墓 | 有力使不出 | 用神地支为墓库，被锁住 |
| 被泄 | 力量分散 | 用神被食伤过度泄气 |
| 虚浮 | 虽有如无 | 用神只在干不透根，空有其表 |

## 四、用神与大运流年的互动

这是分析中最关键的环节。用神判断对否，不在原局，而在岁运的应验。

### 大运对用神的影响

- **用神运**: 十年顺遂，事业进步，身体康健。用神透干之大运 → 功名显达
- **喜神运**: 贵人相助，用神得力。如用神为木，大运行水（水生木）即为喜神运
- **忌神运**: 诸事不顺，灾祸频发。但忌神运中若有制约则无害：
  - 忌神被大运自身干支克制 → 凶中藏吉
  - 忌神被原局用神克制 → 有惊无险
- **用神受制运**: **最凶**。大运恰好克制原局用神 → 事业崩盘、健康出问题
- **仇神运**: 生忌神克用神 → 双重打击

### 用神随大运变化（用神变格）

某些命局用神会随大运改变：

1. **原局用神入墓**: 行墓运则用神不显 → 此阶段需另寻用神
2. **原局太寒**: 走火运调候后，原先前不显的用神现身 → 用神变化
3. **从格逢根运**: 从格本来喜所从之五行，但行运中得一根气 → 由从变不从 → 格局变化
4. **化格遇还原运**: 化格遇还原运 → 化学反应逆转
