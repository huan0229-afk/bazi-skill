# 用神体系

用神是八字中最关键的概念——对命局最有利的那个五行/十神。找准用神，方能准确判断吉凶祸福。

## 一、用神取法四大体系

### 1. 扶抑法（平衡法）— 最常用

日主强旺 → 用神取克泄耗（官杀、食伤、财）
日主衰弱 → 用神取生扶（印、比劫）

**判断日主强弱**:
- 得令（生旺月）：+40%
- 得地（地支通根）：+30%
- 得生（印星贴身）：+20%
- 得助（比劫帮扶）：+10%

**日主强弱等级**:
- **太强**: 用官杀克，或食伤泄，不宜再用印比
- **偏强**: 用财官食伤
- **中和**: 随运变化
- **偏弱**: 用印比
- **太弱（从格）**: 不能扶，顺其势

### 2. 调候法 — 穷通宝鉴体系

寒暖燥湿的平衡。日主生于不同月份，因气候不同而有不同需求。

**调候原则**:
- **冬生（亥子丑月）**: 天寒地冻，急须火调候，其次木生火。水冷木不发，无火则万物不生
- **夏生（巳午未月）**: 火炎土燥，急须水调候，其次金生水
- **辰戌丑未月**: 土当令而燥（未戌）或湿（辰丑），亦须调候
- **春生**: 寒暖交接，视具体命局。春初（寅月）仍须火暖
- **秋生**: 金当令而寒，须火暖金方能成器

**穷通宝鉴逐月调候精要**（详见 classical-texts/qiongtongbaojian.md）:

以日主为甲木为例：
- 寅月甲木：寒木向阳，须火暖之
- 卯月甲木：春木当令，有水水火既济最佳
- 辰月甲木：木有余气，火土并用
- 巳月甲木：夏木枯燥，急须水润
- 午月甲木：炎炎夏日，非水不可
- 未月甲木：土燥木枯，水为先
- 申月甲木：秋金克木，先丁火制金，次水润木
- 酉月甲木：同上，更须火制金
- 戌月甲木：燥土，先水后火
- 亥月甲木：寒木，须火暖
- 子月甲木：极寒，火土并用
- 丑月甲木：寒土，火为先

**各日主通用调候**:
- **甲乙木**: 夏须水，冬须火
- **丙丁火**: 夏须水制其炎，冬须木生其身
- **戊己土**: 夏须水润，冬须火暖
- **庚辛金**: 夏须水淘，冬须火煅
- **壬癸水**: 冬须火防冻，夏须金生水

### 3. 通关法

两种五行相战，取通关之五行为用。

| 相战 | 通关五行 |
|------|----------|
| 木土交战 | 火 |
| 土水交战 | 金 |
| 水火交战 | 木 |
| 火金交战 | 土 |
| 金木交战 | 水 |

> 例：食伤（木）制杀（土）太过 → 用火通关，木生火，火生土，化敌为友

### 4. 病药法 — 神峰通考体系

找出命局中最突出的"病"，用神即为"药"。

- 太旺 → 泄耗为药
- 太弱 → 生扶为药
- 官杀混 → 去其一为药
- 寒 → 火为药
- 燥 → 水为药

## 二、喜神、忌神、仇神、闲神

| 类型 | 定义 | 作用 |
|------|------|------|
| **用神** | 对命局最利的五行 | 扶助用神则吉 |
| **喜神** | 生助用神者 | 喜神出现，用神得力 |
| **忌神** | 克制用神者 | 忌神当运，灾祸立至 |
| **仇神** | 用神克之而反受伤 | 如用神为金，火为忌神，木为仇神（木生火克金） |
| **闲神** | 对用神无显著影响 | 无关紧要 |

---

## 三、用神有力判断

用神须满足以下条件方为有力：

1. **得令**: 用神在月令当旺
2. **得地**: 用神在地支有根（通根）
3. **得生**: 有喜神生助用神
4. **得助**: 用神在天干透出
5. **不受克**: 用神不被忌神克制

> 用神虚浮无根，如纸老虎，虽有如无。
> 用神藏而不透，待大运引出方显其用。

---

## 四、大运与用神

用神在不同大运中的表现：

- **用神运**: 一切顺遂，事业进步，身体健康
- **喜神运**: 贵人相助，用神得力
- **忌神运**: 诸事不顺，灾祸频发。但忌神被制则无害
- **用神受制运**: 最凶，用神被大运或流年克制
