# 阴阳五行基础

## 一、阴阳

### 基本属性

| 阳 | 阴 |
|----|----|
| 天、日、男、动、刚、热、上、左 | 地、月、女、静、柔、冷、下、右 |
| 奇数、明亮、功能、气化 | 偶数、黑暗、物质、成形 |
| 甲丙戊庚壬 | 乙丁己辛癸 |
| 子寅辰午申戌 | 丑卯巳未酉亥 |

### 阴阳关系

- **对立制约**: 阴阳互相制约，维持平衡
- **互根互用**: 孤阴不生，独阳不长
- **消长转化**: 阴极阳生，阳极阴生（冬至一阳生，夏至一阴生）
- **阴阳交错**: 阳中有阴（午藏丁己），阴中有阳（子藏癸）

### 八字中阴阳的意义

日主分阴阳，决定：
- 十神细分（同性为偏、异性为正）
- 命主性情（阳主动外向、阴主静内向）
- 大运顺逆（阳男阴女顺排、阴男阳女逆排）

### 五行相生

```
木 → 火 → 土 → 金 → 水 → 木
```

- **木生火**: 木燃烧生火。木旺则火得源，木衰则火无继
- **火生土**: 火焚物成灰土。火旺则土得母，火衰则土失生
- **土生金**: 土中蕴藏金石。土旺则金得生，土衰则金失养
- **金生水**: 金熔化为液体。金旺则水有源，金衰则水失本
- **水生木**: 水灌溉生木。水旺则木得润，水衰则木枯槁

### 五行相克

```
木 → 土 → 水 → 火 → 金 → 木
```

- **木克土**: 树木破土而出，木旺则土虚
- **土克水**: 土筑堤防水，土旺则水滞
- **水克火**: 水能灭火，水旺则火熄
- **火克金**: 烈火熔金，火旺则金销
- **金克木**: 斧斤伐木，金旺则木伤

### 亢乘与反侮

- **亢乘**: 主克方过强，被克方过弱 → 被克方受损严重
- **反侮**: 被克方过强反而克制主克方 → 木坚金缺、水盛土溃、火炎水干、土重木折、金多火熄

### 五行旺相休囚死（四季旺衰）

| 木 | **旺** | 休 | 死 | 相 | 囚 |
| 火 | 相 | **旺** | 囚 | 死 | 休 |
| 土 | 死 | 相 | 休 | 囚 | **旺** |
| 金 | 囚 | 死 | **旺** | 休 | 相 |

> **旺**: 当令之气，力量最强；**相**: 旺气所生，次旺；**休**: 生旺气者，退休；**囚**: 克旺气者，被囚；**死**: 被旺气克者，最弱

> **记忆规律**: 
> - **阳干**（甲丙戊庚壬）顺行十二支（子→丑→寅...）
> - **阴干**（乙丁己辛癸）逆行十二支（子→亥→戌...）
> - **长生位**: 甲木长生在亥（水能生木）、丙戊长生在寅（木能生火）、庚长生在巳（火生土，土生金）、壬长生在申（金生水）；乙长生在午、丁己长生在酉、辛长生在子、癸长生在卯

**十二长生的核心应用**:

1. **判断天干是否有根气**: 天干在地支处于长生/冠带/临官/帝旺/墓 → 有根（其中临官=禄，帝旺=旺，力量最强；墓库次之）
2. **八字无根则虚浮**: 天干在地支无长生/禄/旺/库 → 虚浮无力，虽有如无
3. **判断旺衰转折**: 从长生到帝旺为上升期，从衰到绝为下降期
4. **看大运时机**: 日主进入禄旺之运必有大发展；入死绝之运多困厄
5. **自坐墓库**: 日支为日主墓库，主性格内向、隐忍、不轻易表露

### 五行的生克辩证

**生非全吉，克非全凶**:

- **生多为克**: 生扶过旺反成克制。如木多火塞（木太重反而压熄火）、金多水浊（金太重反而搅浑水）
- **克中有生**: 金克木 → 木得金修剪方能成材；土克水 → 水得土堤方不泛滥；火克金 → 金得火炼方成器皿
- **相生不一定有情**: 印星生身，但印多则主人依赖、无主见、懒惰
- **相克不一定为害**: 官杀克身，有制则主权贵；食伤泄身，不过则有才华

**五行生克的层次**:

1. **天干之生克**: 表象，外显，直接影响运势（君）
2. **地支之生克**: 内在，根基，通过刑冲合害表现（臣）
3. **藏干之生克**: 暗藏，隐晦，通过透干方能显现（民）

### 滴天髓论五行

> 木：木性腾上而无所止，气重则金，使木得金而为用。
> 火：火炎上而性热，得水方成既济。
> 土：土重而滞，得水则润，得木则疏。
> 金：金得火方成器，无火则顽。
> 水：水性润下，得土则安，水泛则木浮。

**滴天髓五行精义扩展**:

> 五气者，五行之气也。五气偏全，而定人之吉凶。
> 五行和者，一世无灾。五行偏枯，终身有疾。
> 太过者，宜泄宜克；不及者，宜生宜扶。
> 上下贵乎情协，左右贵乎气协。情协者，干支配合有情；气协者，各宫五行协调。

## 三、五行在八字中的应用

1. **日主五行**决定命主基本属性（性格、体质、思维模式）
2. **五行平衡**是判断命局好坏的第一标准（偏枯为病，中和为贵）
3. **五行的旺衰组合**决定用神方向（旺者宜泄耗、弱者宜生扶）
4. **五行对应**可断行业方向、健康问题、人际关系
5. **十二长生**判断各天干在地支中的真实力量，比单纯"生克"更精确
6. **生克辩证**：生多亦为克，克中有生机，不可机械论生克

### 日主强弱快判法

1. **看月令**: 日主在月令处于什么状态？（旺/相/休/囚/死）
2. **看长生**: 日主在地支中是否有禄位（临官）或旺位（帝旺）？
3. **看库根**: 日主是否有墓库为根？（虽弱但有根）
4. **看生扶**: 天干有无印星、比劫贴身？
5. **综合判断**: 月令旺 + 地支有根 = 身强；月令囚 + 地支无根 = 身弱
