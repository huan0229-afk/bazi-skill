# 神煞大全

神煞是八字辅助参考，**不可喧宾夺主**。十神格局为主，神煞为佐。神煞可佐证但不能改变十神格局的判断。

## 使用原则

1. 十神和格局决定命局层次 → 神煞只是点缀
2. 吉神不为忌，凶煞不为喜 → 吉神逢生扶方吉，凶煞被制方无害
3. 天乙贵人最尊 → 逢之可解百凶
4. 神煞忌多 → 多则不灵

## 一、吉神

### 1. 天乙贵人

**口诀**: 甲戊庚牛羊（丑未），乙己鼠猴乡（子申），丙丁猪鸡位（亥酉），壬癸兔蛇藏（巳卯），辛马虎相从（午寅）

**查法**: 以日干为主（或年干为辅），查四柱地支。丑未为甲戊庚之天乙贵人，余类推。

**意义**: 遇难有救，化险为夷，主贵人提携，地位崇高。天乙为诸贵人之首，最为尊贵。

### 2. 太极贵人

**口诀**: 甲己子午中，乙庚丑未寻，丙辛寅申上，丁壬卯酉真，戊癸辰戌是

**查法**: 以日干查四柱地支。

**意义**: 智慧超群，有特殊才华，好神秘学、哲学、术数。

### 3. 文昌贵人

**口诀**: 甲巳乙午丙戊申，丁己酉位庚亥乡，辛子壬寅癸卯位

**查法**: 以日干查四柱地支。

**意义**: 文学才华，考试运佳，学术研究有成就。文昌入命主文采风流。

### 4. 天德贵人

**口诀**: 正丁二申三壬四辛五亥六甲，七癸八寅九丙十乙子巳丑庚
（正月生人见丁为天德，二月见申，三月见壬...）

**意义**: 天德为月令之德神，逢凶化吉，消灾解难。是化解灾祸最强的吉星。

### 5. 月德贵人

**口诀**: 寅午戌月德在丙，申子辰月德在壬，亥卯未月德在甲，巳酉丑月德在庚

**意义**: 月德与天德并称"天月二德"，二德俱全为上格。

### 6. 禄神

**口诀**: 甲禄在寅，乙禄在卯，丙戊禄在巳，丁己禄在午，庚禄在申，辛禄在酉，壬禄在亥，癸禄在子

**意义**: 禄神代表福禄、薪俸、财富基础。禄神被冲则破财失业。

### 7. 金舆

**口诀**: 甲龙乙蛇丙戊羊，丁己猴位庚犬乡，辛猪壬牛癸寻虎

**意义**: 金舆为车马星，主富贵得车马。现代为有车代步、交通便利。

## 二、凶煞

### 1. 羊刃

**口诀**: 甲刃在卯，乙刃在寅，丙戊刃在午，丁己刃在巳，庚刃在酉，辛刃在申，壬刃在子，癸刃在亥

**意义**: 羊刃为刀锋，主性暴、血光、手术、克妻。**羊刃无制为大凶**。有七杀可制羊刃，反主武将之贵。

### 2. 劫煞

**口诀**: 寅午戌见亥，申子辰见巳，巳酉丑见寅，亥卯未见申

**意义**: 劫煞主动荡破财，遭小人暗算。

### 3. 灾煞

**口诀**: 寅午戌见子，申子辰见午，巳酉丑见卯，亥卯未见酉

**意义**: 灾煞主突来之灾、意外。被冲则更凶。

### 4. 亡神

**口诀**: 寅午戌见巳，申子辰见亥，巳酉丑见申，亥卯未见寅

**意义**: 亡神主动荡不安，心神不宁，易有精神类疾病。

---

### 5. 孤辰寡宿

**口诀**: 寅卯辰见巳为孤辰、丑为寡宿；巳午未见申为孤辰、辰为寡宿；申酉戌见亥为孤辰、未为寡宿；亥子丑见寅为孤辰、戌为寡宿

**意义**: 孤辰主性格孤僻、不喜社交；寡宿主六亲缘薄、独居倾向。男怕孤辰，女怕寡宿。

### 6. 丧门吊客

**丧门**: 子年见寅、丑年见卯...顺推三位
**吊客**: 子年见戌、丑年见亥...逆推一位

**意义**: 流年遇丧门吊客，主近亲有伤病或去世。丧门重孝服，吊客重慰问。

### 7. 血刃（血支）

**口诀**: 寅午戌见丑、申子辰见卯、巳酉丑见未、亥卯未见酉

**意义**: 血刃主血光、手术、意外伤害。逢冲或临羊刃则更验。

### 8. 勾绞煞

**口诀**: 以年支三合局看：寅午戌见丑为勾、卯为绞；申子辰见未为勾、酉为绞；巳酉丑见辰为勾、亥为绞；亥卯未见戌为勾、子为绞

**意义**: 勾绞主口舌是非、官非纠纷、被牵连。逢凶神引动则易打官司。

## 三、中性神煞

### 1. 桃花（咸池）

**口诀**: 寅午戌见卯（沐浴在卯），申子辰见酉，巳酉丑见午，亥卯未见子

**意义**: 桃花主人缘、异性缘、情欲。桃花在四柱不同位置意义不同：
- 年桃花：早熟或家中有艺术气息
- 月桃花：青年桃花旺，异性朋友多
- 日桃花：配偶漂亮，婚姻多情事
- 时桃花：晚年仍有异性缘，或子女风流

桃花被冲则感情变动，桃花被合则情缘深重。

### 2. 驿马

**口诀**: 寅午戌见申，申子辰见寅，巳酉丑见亥，亥卯未见巳

**意义**: 驿马主奔波、移动、旅行、搬迁。驿马逢冲主大变动；驿马逢财主走动中得财。

### 3. 华盖

**口诀**: 寅午戌见戌，申子辰见辰，巳酉丑见丑，亥卯未见未

**意义**: 华盖主孤独、清高、有才华但与众不同。华盖重者好宗教玄学，有艺术天赋但孤芳自赏。

### 4. 将星

**口诀**: 寅午戌见午，申子辰见子，巳酉丑见酉，亥卯未见卯

**意义**: 将星为权柄之星，主领导才能，适合管理岗位。将星入命有指挥之才。

### 5. 学堂

**口诀**: 甲见巳、乙见午、丙见寅、丁见戌、戊见寅、己见辰、庚见申、辛见未、壬见申、癸见辰

**意义**: 学堂主学业、考运。学堂+文昌俱全，主高学历、考试运佳。

### 6. 天厨贵人

**口诀**: 甲见巳、乙见午、丙见子、丁见巳、戊见午、己见申、庚见寅、辛见午、壬见酉、癸见亥

**意义**: 天厨主食禄丰盛、口福佳、宜饮食行业。入命主不愁吃穿。

### 7. 红鸾天喜

**红鸾**: 子见卯、丑见寅、寅见丑、卯见子、辰见亥、巳见戌、午见酉、未见申、申见未、酉见午、戌见巳、亥见辰
**天喜**: 红鸾对冲之位（子见酉、丑见申...）

**意义**: 红鸾主婚缘、添丁之喜；天喜主喜庆吉事。流年遇红鸾天喜，婚事、生子概率大增。

### 8. 福星贵人

**口诀**: 甲见丑、乙见寅、丙见子、丁见酉、戊见申、己见未、庚见午、辛见巳、壬见辰、癸见卯

**意义**: 福星入命一生少灾，平平安安。不如天乙尊贵，但更接地气，主安稳。

## 四、神煞组合判断

| 组合 | 意义 |
|------|------|
| 天乙+文昌 | 贵人相助学业事业 |
| 桃花+驿马 | 情缘在旅行中，或异地恋 |
| 华盖+文昌 | 学者型人才 |
| 羊刃+七杀 | 武将之贵 |
| 禄神+驿马 | 工作多变动但收入不差 |
| 桃花+羊刃 | 因色致祸 |
| 天月二德俱全 | 一生少灾，逢凶化吉 |

## 渊海子平论神煞

> 神煞者，吉凶之助也。吉神遇之愈吉，凶神遇之愈凶。
> 然吉神不可多，多则不验。凶神不可无救，无救则祸烈。
