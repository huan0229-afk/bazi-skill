## 二、五行失衡与疾病

### 五行过旺

- **木过旺**: 肝阳上亢 → 高血压、头痛、易怒、中风
- **火过旺**: 心火亢盛 → 心悸、失眠、口腔溃疡、炎症
- **土过旺**: 脾胃壅滞 → 肥胖、消化不良、倦怠
- **金过旺**: 肺气壅塞 → 哮喘、皮肤干燥、便秘
- **水过旺**: 肾水泛滥 → 水肿、尿频、畏寒、腰痛

### 五行过弱

- **木过弱**: 肝血不足 → 头晕、近视、筋骨无力
- **火过弱**: 心气不足 → 怕冷、贫血、精神萎靡
- **土过弱**: 脾胃虚弱 → 消化不良、面黄肌瘦
- **金过弱**: 肺气不足 → 易感冒、气短、皮肤差
- **水过弱**: 肾精不足 → 腰酸、耳鸣、早衰

## 三、十神与健康

### 致病模式

| 十神组合 | 健康问题 |
|----------|----------|
| 枭神夺食 | 消化系统疾病、意外伤害。食神主饮食消化，被偏印夺则饮食不纳 |
| 伤官见官 | 神经衰弱、精神压力。伤官乃耗泄之神，逢官则压力倍增 |
| 七杀攻身（无制） | 外伤、手术、暴力伤害。七杀无制则克身太过 |
| 财多身弱 | 积劳成疾，多因过度劳累致病 |
| 印星过旺 | 代谢迟缓、肥胖、迟钝 |

### 十神在四柱与健康

- **年柱**: 头部、幼年体质
- **月柱**: 胸部、呼吸、消化系统
- **日柱**: 腹部、腰部、生殖系统
- **时柱**: 下肢、晚年健康

## 四、地支冲刑与疾病

- **子午冲**: 水火相冲 → 心血管、泌尿系统
- **卯酉冲**: 金木相冲 → 肝胆、呼吸系统
- **寅申冲**: 金木相冲+驿马冲 → 车祸、骨折
- **巳亥冲**: 火水相冲 → 炎症反复
- **丑未冲、辰戌冲**: 脾胃疾病

**相刑**: 自刑（辰辰、午午、酉酉、亥亥）主自我伤害、自残倾向，或慢性反复发作疾病

## 五、神煞与伤病

- **羊刃**逢冲：手术、血光、意外
- **七杀**无制：外伤、暴力伤害
- **丧门吊客**入命：近亲有难
- **灾煞**临太岁：突来之灾
- **血刃**入命：易有手术

## 六、大运与健康转折

- **用神运**: 身体健康，精力充沛
- **忌神运**: 旧病复发，体质下降
- **财破印运**: 注意长辈健康（印为母）
- **枭神夺食运**: 注意自身饮食消化、意外
- **伤官见官运**: 精神压力大，失眠焦虑

### 生死关口判断（需非常慎重）

以下组合为危险信号，但需综合全盘判断，不可轻断生死：
- 用神被大运流年同时克制
- 日主之根被拔（地支全冲）
- 禄神被冲
- 七杀+羊刃+灾煞联动
