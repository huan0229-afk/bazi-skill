# 大运流年分析

## 一、排大运法

### 起运岁数

从出生日到下一个（或上一个）节气的天数除以3，即为起运岁数。

- **阳年男**（甲丙戊庚壬年出生的男性）: 顺排，数到**下一个节**
- **阴年女**（乙丁己辛癸年出生的女性）: 顺排，数到**下一个节**
- **阴年男**: 逆排，数到**上一个节**
- **阳年女**: 逆排，数到**上一个节**

### 大运推算

以月柱为起点，顺排则每步大运干支向前推一位（干支各进一），逆排则向后推一位。

每步大运管十年。

### 标准的"起运"年龄

起运岁数多为：1、2、3、4、5、6、7、8、9 岁起运。
（部分为 0 岁起运，即出生即上运）

## 二、大运解读

### 大运干支各管五年

- **天干前五年**: 大运天干的影响力在前五年最大
- **地支后五年**: 大运地支的影响力在后五年最大
- 干支配合看，不可偏废

### 大运与原局的关系

- **大运是原局的延伸**: 原局好的方面在大运中得以体现
- **原局有，大运无**: 待时而发
- **原局无，大运来**: 运过则消

### 大运的吉凶判断

**大运对用神的关系**:
- 大运是用神 → 十年好运
- 大运是喜神 → 助用神，亦吉
- 大运是忌神 → 十年困顿，但需看是否被制
- 大运克用神 → 大凶，事业受阻
- 大运用神被合 → 用神失其用，先吉后凶

**大运干支上下关系**:
- 干为喜，支为忌：前五年吉，后五年凶
- 干为忌，支为喜：前五年凶，后五年吉
- 干支皆喜：十年全吉
- 干支皆忌：十年全凶
- 地支生天干（喜神生用神）：喜上加喜
- 天干克地支（忌神制用神）：凶中更凶

### 换运（交运）

每到换运之年（如7岁起运，则17、27、37...岁），运势发生方向性变化，往往伴随人生重大事件（搬家、换工作、婚变等）。

## 三、流年分析

### 太岁

流年地支为太岁，为一年之主。太岁不可冲，冲则必有变动。

**太岁的三条铁律**:
1. **太岁不可犯**: 日柱冲太岁 → 自身变动；年柱冲太岁 → 家庭根基动
2. **太岁为君**: 大运为臣，流年为君。大运吉流年凶，不为大凶；大运凶流年吉，难有大喜
3. **太岁入卦**: 流年干支进入原局，与原局发生合冲刑害关系

### 流年与大运原局互动法则

#### 天克地冲（岁运并见）

大运与流年天干相克+地支相冲，或流年与原局某柱天克地冲：

- **流年冲大运**: 变动激烈，好事坏事都加倍。吉则大喜，凶则大凶
- **流年冲年柱**: 祖上、家庭、父母有变动
- **流年冲月柱**: 工作、事业、父母有大变动
- **流年冲日柱**: 自身、配偶有变动（婚变、搬家、跳槽）— **最验**
- **流年冲时柱**: 子女、下属、投资有变动

**天克地冲的判断**:
- 冲喜神（用神的根被冲）→ 大凶，事业根基动摇
- 冲忌神（忌神被冲走）→ 反而为吉，坏事消除
- 冲合局（三合局被冲散）→ 团队解散、合约破裂

#### 岁运并临

大运与流年干支完全相同时为"岁运并临"：
> 岁运并临，吉凶倍增。运吉者当年大吉，运凶者当年大凶。

- 岁运并临为用神 → 十年好运集中一年爆发，飞黄腾达
- 岁运并临为忌神 → 十年坏运集中于一年，灾难深重
- 岁运并临+流年又冲日柱 → 必有生命危险或重大变故

#### 伏吟返吟

- **伏吟**: 流年与某柱干支相同（如原局有甲子，流年又见甲子）→ 该柱所主之事有重叠、反复、停滞
- **返吟**（天克地冲）: 流年与原局某柱天干相克+地支相冲 → 变动、冲突、分离
- 伏吟主"静"，返吟主"动"。返吟应事比伏吟快且猛

#### 合冲交加

- **先合后冲**: 先成后破。婚姻合同先谈好了又破裂
- **先冲后合**: 先破后成。分手后复合，破裂后重谈
- **三合局全之年**: 五行力量偏向某方，该五行相关事项集中发生
- **三会局全之年**: 更强烈的五行偏向，大事件发生年

### 流年应期十大法则

1. **用神到位之年**: 事业进步，婚缘出现，贵人提携
2. **忌神到位之年**: 百事不顺，灾祸频发
3. **夫妻宫被合之年**: 婚缘信号（未婚者结婚，已婚者情缘变动）
4. **夫妻宫被冲之年**: 婚姻危机信号（争吵、分居、离婚）
5. **官星/财星到位之年**: 婚缘或事业上升信号
6. **伤官见官之年**: 女命婚姻危机，男命官非口舌
7. **枭神夺食之年**: 健康问题、意外、食欲不振
8. **财星破印之年**: 为财失志、长辈健康问题、学业挫折
9. **比劫夺财之年**: 破财、被分财、投资失败、克妻（男）
10. **财库被冲之年**: 暴富或暴贫，财务大变动

### 流月流日

流年为基础，流月流日只起催动作用。月令与日柱相冲时，当月当周注意变动。

**流月判断法则**:
1. **流月冲日支**: 当月情感波动、家庭纠纷
2. **流月与流年天克地冲**: 当年凶运在此月集中爆发
3. **流月三合太岁**: 当月有好事，运势上升
4. **流月入原局**: 流月干支恰好是原局的某柱 → 当月应验该柱所代表之事

**流月+十神**: 流月天干十神 → 当月主题；流月地支与原局互动 → 具体事件

**流日**: 一般只作参考。日柱与流日相冲时，当天可能变动。

### 大运与原局各柱的关系

大运干支可以与原局任何一柱发生互动：

- **大运为年柱的应期**: 大运干支进入年柱或与年柱互动 → 祖上、家庭、父母之事
- **大运为月柱的应期**: 大运干支进入月柱或与月柱互动 → 事业、工作、父母之事
- **大运为日柱的应期**: 大运干支进入日柱或与日柱互动 → 自身、配偶之事 **（最直接）**
- **大运为时柱的应期**: 大运干支进入时柱或与时柱互动 → 子女、投资、晚年之事

**大运与原局的五种互动方式**:
1. **穿（进入）**: 大运干支与某柱相同 → 直接进入该柱，该柱之事凸显
2. **合**: 大运干支合某柱 → 合则有情，事有牵连
3. **冲**: 大运冲某柱 → 冲则变动，该柱之事不稳定
4. **刑**: 大运刑某柱 → 刑则有伤，该柱之事有烦恼
5. **生**: 大运天干生某柱天干 → 生则扶持，该柱之事顺遂

### 整个人生运势曲线

一个人的一生可分为几个阶段：

- **少年运（0-20岁）**: 看年柱、第一二步大运，看家庭、学业
- **青年运（20-40岁）**: 看月柱、第三四步大运，看事业起步、婚姻
- **中年运（40-60岁）**: 看日柱、第五六步大运，看事业巅峰、财富
- **晚年运（60岁+）**: 看时柱、第七八步大运，看退休、健康

### 运势类型

- **步步高升型**: 大运连续为喜用神，一运好过一运
- **先苦后甜型**: 前半生忌神运，后半生用神运
- **先甜后苦型**: 前半生用神运，后半生忌神运
- **起伏不定型**: 喜忌交替，大起大落

## 滴天髓论运

> 运之所在，吉凶随之。运过则仍旧。
> 人有凌云之志，非运不能自达。运去黄金失色，时来顽铁生光。
> 大运者，十年之吉凶。吉运虽然吉，凶神制则无害；凶运虽然凶，吉神救则无忧。
