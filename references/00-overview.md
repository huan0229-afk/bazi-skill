## 分析原则（重要！）

分析任何八字必须遵循以下顺序，不可跳跃：

1. **日主强弱先判** — 得令？得地？得生？得助？日主强弱决定一切后续判断
2. **格局后定** — 以月令主气定格局，看是否有情有力
3. **用神再取** — 格局定后用神方有方向：扶抑、调候、通关、病药
4. **大运看时应** — 用神在何运得力？忌神在何运被制？
5. **神煞最后补** — 神煞只能辅助，绝不能凌驾于十神格局之上

**看性格**: 日主十干性情 + 十神组合 + 五行平衡度
**看事业**: 格局类型 + 官杀印星 + 财星组合 + 大运方向
**看财富**: 财星旺衰 + 食伤源头 + 日主是否能担 + 财库状态
**看婚姻**: 配偶星（男财女官）+ 夫妻宫（日支）+ 桃花神煞

## 常用术语速查

**基础概念**:
- **日主/日元**: 日柱天干，代表命主本人
- **格局**: 以月令为主的八字结构分类
- **用神**: 对八字最有利的五行/十神
- **喜神**: 生助用神者
- **忌神**: 克制用神者
- **仇神**: 生忌神克用神者
- **闲神**: 对命局无显著影响者

**干支状态**:
- **透干**: 藏干在天干出现 → 力量显豁
- **通根**: 天干在地支有同类藏干 → 根基扎实
- **得令**: 日主在月令处于旺相状态 → 力量40%
- **得地**: 日主在地支有禄/旺/长生/库 → 力量30%
- **得生**: 有印星生扶日主 → 力量20%
- **得助**: 有比劫帮身 → 力量10%

**格局术语**:
- **成格**: 格局成立，条件具备
- **败格**: 格局被破坏
- **救应**: 格局虽败但有救 → 仍可为贵
- **变格**: 从格、化格、专旺格等非正格
- **清纯**: 格局不杂不乱
- **混杂**: 官杀、食伤、印枭等同现

**岁运术语**:
- **大运**: 十年一换的运势周期
- **流年**: 当年的运势（太岁）
- **起运**: 开始上运的年龄
- **交运**: 换大运的年份
- **岁运并临**: 大运与流年干支相同 → 吉凶加倍
- **天克地冲**: 天干相克+地支相冲 → 变动剧烈
- **伏吟**: 干支重复出现 → 事有重复
- **返吟**: 天克地冲的别名 → 事有反复
