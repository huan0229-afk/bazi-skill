# 婚姻感情专题

## 一、配偶定位

### 男命
- **正财**为妻星（正妻）
- **偏财**为妾星（情人、短暂关系）
- 日支为妻宫

### 女命
- **正官**为夫星（正夫）
- **七杀**为偏夫（情人、短暂关系）
- 日支为夫宫

---

## 二、配偶宫（日支）分析

日支与日干的关系直接反映夫妻互动模式：

| 日支十神 | 夫妻关系 |
|----------|----------|
| 比肩 | 配偶独立，夫妻如朋友，但也易争执 |
| 劫财 | 争吵多，经济分歧。男命克妻 |
| 食神 | 配偶体贴，夫妻和睦温馨 |
| 伤官 | 冲突多，互相挑剔。女命最忌 |
| 正财 | 男命得贤妻，夫妻互助 |
| 偏财 | 男命妻家富裕或娶年少者 |
| 正官 | 女命得贵夫，婚姻幸福 |
| 七杀 | 女命夫刚强，婚姻多争执 |
| 正印 | 男命妻如母，平和但欠激情 |
| 偏印 | 夫妻个性不合，沟通不畅 |

### 日支逢冲

日支被冲（月支冲、时支冲、大运冲）：婚姻**必有大变动**，或争吵、分居、离婚。

## 三、婚期判断

### 早婚信号
- 财官星在年月早现
- 桃花在年月柱
- 日支合月支或年支
- 20岁前大运见官星（女）/财星（男）

### 晚婚信号
- 财官星在日时晚现
- 华盖、孤辰、寡宿入命
- 伤官旺（女命）
- 比劫旺（男命）

### 婚期应期
以下年份易结婚：
- **日支（夫妻宫）被合之年**
- **官星（女）/财星（男）透出之年**
- **桃花星出现之年**
- **红鸾天喜星入命之年**

---

## 四、婚姻不顺的信号

### 男命克妻
1. **比劫林立**：夺财克妻
2. **财星弱而虚浮**：无妻或妻不能助
3. **财星入墓或逢冲**：妻健康有问题
4. **日支为劫财或七杀**：夫妻关系紧张

### 女命克夫
1. **伤官旺而贴近日干**：克正官。伤官见官，为祸百端
2. **七杀混杂**（正官七杀同现）：夫星不明，感情复杂
3. **日支为伤官或劫财**：夫妻不和
4. **官星弱而受制**：夫缘薄或夫无能

### 双方问题
- **日支逢冲**: 婚姻不稳定
- **桃花泛滥**: 婚外情风险高
- **阴阳差错日**: 甲午、丙子、丁丑、戊寅、庚辰、辛巳、壬申、癸酉等日，婚姻多波折

---

## 五、合婚要则

1. **五行互补**: 一方八字缺的五行另一方旺 → 互补
2. **日柱相合**: 甲己合、乙庚合等日干五合 → 天生吸引
3. **年柱相生**: 年生年、年支生年支 → 家庭和谐
4. **忌**: 日柱对冲（寅申、巳亥、子午、卯酉） → 天生不合
5. **忌**: 双方的忌神相同 → 同病不相济

> 财星入墓，妻多病。官星落空，夫难靠。
> 桃花若临官位，女命须防情劫。
