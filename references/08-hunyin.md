# 婚姻感情专题

## 一、配偶定位

### 男命
- **正财**为妻星（正妻）
- **偏财**为妾星（情人、短暂关系）
- 日支为妻宫

### 女命
- **正官**为夫星（正夫）
- **七杀**为偏夫（情人、短暂关系）
- 日支为夫宫

---

## 二、配偶宫（日支）分析

日支与日干的关系直接反映夫妻互动模式：

| 日支十神 | 夫妻关系 |
|----------|----------|
| 比肩 | 配偶独立，夫妻如朋友，但也易争执 |
| 劫财 | 争吵多，经济分歧。男命克妻 |
| 食神 | 配偶体贴，夫妻和睦温馨 |
| 伤官 | 冲突多，互相挑剔。女命最忌 |
| 正财 | 男命得贤妻，夫妻互助 |
| 偏财 | 男命妻家富裕或娶年少者 |
| 正官 | 女命得贵夫，婚姻幸福 |
| 七杀 | 女命夫刚强，婚姻多争执 |
| 正印 | 男命妻如母，平和但欠激情 |
| 偏印 | 夫妻个性不合，沟通不畅 |

### 日支逢冲

日支被冲（月支冲、时支冲、大运冲）：婚姻**必有大变动**，或争吵、分居、离婚。

---

## 三、婚期判断

### 早婚信号
- 财官星在年月早现
- 桃花在年月柱
- 日支合月支或年支
- 20岁前大运见官星（女）/财星（男）

### 晚婚信号
- 财官星在日时晚现
- 华盖、孤辰、寡宿入命
- 伤官旺（女命）
- 比劫旺（男命）

### 婚期应期
以下年份易结婚：
- **日支（夫妻宫）被合之年**
- **官星（女）/财星（男）透出之年**
- **桃花星出现之年**
- **红鸾天喜星入命之年**

---

## 四、婚姻不顺的信号

### 男命克妻
1. **比劫林立**：夺财克妻
2. **财星弱而虚浮**：无妻或妻不能助
3. **财星入墓或逢冲**：妻健康有问题
4. **日支为劫财或七杀**：夫妻关系紧张

### 女命克夫
1. **伤官旺而贴近日干**：克正官。伤官见官，为祸百端
2. **七杀混杂**（正官七杀同现）：夫星不明，感情复杂
3. **日支为伤官或劫财**：夫妻不和
4. **官星弱而受制**：夫缘薄或夫无能

### 双方问题
- **日支逢冲**: 婚姻不稳定
- **桃花泛滥**: 婚外情风险高
- **阴阳差错日**: 甲午、丙子、丁丑、戊寅、庚辰、辛巳、壬申、癸酉等日，婚姻多波折

---

## 五、合婚要则

1. **五行互补**: 一方八字缺的五行另一方旺 → 互补
2. **日柱相合**: 甲己合、乙庚合等日干五合 → 天生吸引
3. **年柱相生**: 年生年、年支生年支 → 家庭和谐
4. **忌**: 日柱对冲（寅申、巳亥、子午、卯酉） → 天生不合
5. **忌**: 双方的忌神相同 → 同病不相济

---

## 六、离婚与婚变信号

离婚或婚姻大变动往往需要多重信号叠加：

**离婚高风险组合**:
1. **日支逢冲** + 伤官见官流年（女命）→ 必有大争吵，离婚概率极高
2. **日支逢冲** + 比劫夺财流年（男命）→ 克妻信号
3. **夫妻宫被合且合化为忌神** → 配偶有外情
4. **原局官杀混杂** + 大运再见官杀（女命）→ 感情混乱期
5. **原局比劫夺财** + 大运再行比劫（男命）→ 婚姻破裂期

**婚变应期三要素必须齐备**:
1. 原局有婚姻不顺的病根
2. 大运引动这个病根
3. 流年触发（日支被冲、官/财星被合走）

> 只有一项不必惊慌，两项叠加宜注意，三项齐备则大概率应验。

### 配偶大致方向

**配偶的五行特征**:
- 男命以财星五行断妻性格：正财为木 → 妻正直仁爱；正财为金 → 妻刚强果断
- 女命以官星五行断夫性格：正官为火 → 夫热情主动；正官为水 → 夫智慧深沉

**配偶的远近**:
- 财官在年月 → 配偶为同乡、同事、身边人
- 财官在日时 → 配偶为远方人、异地相识
- 配偶星在年支 → 童年即认识（青梅竹马）
- 配偶星在时柱 → 晚婚，配偶年纪差距大

**配偶的家境**:
- 财官星为用神且旺 → 配偶家境好，对自己有帮助
- 财官星为忌神 → 配偶家境一般，婚姻拖累自己
- 日支为喜用 → 婚后运势上升，配偶是自己贵人
- 日支为忌神 → 婚后运势下降，婚姻是负担

## 七、三命通会论婚姻

> 男命财星清透，主得贤妻。女命官星得位，主嫁贵夫。
> 财星入墓，妻多病。官星落空，夫难靠。
> 桃花若临官位，女命须防情劫。

### 渊海子平

> 女命伤官福不真，无财无印守孤贫。
> 男命比肩多者，克妻。劫财旺者，争妻。
