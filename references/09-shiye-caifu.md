# 事业财运专题

## 一、事业类型判断

以十神组合定事业方向：

| 十神组合 | 适合行业 | 典型职业 |
|----------|----------|----------|
| 正官+正印 | 公职、行政、教育 | 公务员、教师、法官 |
| 七杀+食神 | 军警、武术、竞争行业 | 军人、警察、运动员 |
| 七杀+印星 | 掌权管理 | 企业管理、政治家 |
| 食神+财星 | 技艺、实业 | 厨师、工匠、实业家 |
| 伤官+财星 | 创新、商业 | 企业家、投资人、发明家 |
| 伤官+印星 | 文艺、学术 | 作家、学者、设计师 |
| 财星+官星 | 金融、政商 | 银行家、财务总监 |
| 比劫+食伤 | 合伙生意、技术劳务 | 工作室、技术工人 |
| 偏印+七杀 | 偏门、特殊技能 | 科研、医生、术数 |

## 二、官运判断

### 官贵信号
1. **正官格成格**（官印相生或财生官）
2. **七杀有制化**（食神制杀或杀印相生）
3. **财官双美**（财生官有力）
4. **伤官佩印**（才华加权力）

### 官运应期
- 官星到位之年 → 升职
- 印星到位之年 → 得权
- 财生官之年 → 事业扩展

### 官非信号
- 伤官见官之年 → 官非口舌
- 七杀无制之年 → 受人攻击
- 官杀混杂无制 → 是非缠身

---

## 三、财运判断

### 富命信号
1. **财星当令有源**（食伤生财）
2. **财气通门户**（财星透干通根）
3. **身财两旺**（日主强财星也强）
4. **食神生财格成**（技艺致富）
5. **从财格**（满盘财星）
6. **财库**（辰戌丑未）被冲开之年暴富

### 贫命信号
1. **财星弱而受克**
2. **比劫夺财无制**
3. **财多身弱**（富屋贫人，财来财去）
4. **财星入墓**（财被锁住）

### 财运应期
- **食伤生财年** → 收入增长、投资收益
- **财库被冲年** → 突然的财富变化（可吉可凶）
- **比劫夺财年** → 破财、被人分财
- **财破印年** → 为财失学、失志、长辈问题

## 四、偏财与投机运

偏财旺者适合商业投资。条件：
1. 偏财透干通根
2. 身强能担财
3. 食伤生财有力
4. 无劫财夺财

**投机暴富**: 偏财+驿马+食神生财 → 敢于冒险的商业奇才

**投机破产**: 偏财+劫财+七杀 → 赌性大但运气差

## 五、合伙与独立

| 十神特征 | 适合模式 |
|----------|----------|
| 比肩旺 | 合伙，与人合作共赢 |
| 劫财旺 | 不宜合伙，易被合伙人算计 |
| 食伤旺 | 独立创业，技术立身 |
| 官星旺 | 适合在大机构内发展 |
| 偏印旺 | 独立研究或专业技术 |

## 六、职业生涯阶段

结合大运看：
- **第一步大运**（0-15岁）：学业运，印星为用则学业佳
- **第二步大运**（15-25岁）：求学或事业起步
- **第三步大运**（25-35岁）：事业上升期，官星/财星为用则飞黄腾达
- **第四步大运**（35-45岁）：事业高峰期
- **第五步大运**（45-55岁）：事业巅峰或转折
- **第六步大运**（55-65岁）：退居二线或二次创业
- **第七八步**（65岁+）：退休运
